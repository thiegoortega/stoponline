# Stop Online Pro

Jogo Stop online multiplayer, pronto para rodar no navegador.

## Como jogar

1. Abra o link do GitHub Pages.
2. Digite seu nome para entrar.
3. Aguarde outros jogadores na tela de espera.
4. O host inicia a rodada.
5. Complete as categorias (Nome, País, Animal, Objeto) antes do timer acabar.
6. Veja o ranking em tempo real.
7. Clique "Próxima Rodada" para continuar jogando.
