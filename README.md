# Stop Online Pro

Jogo Stop online multiplayer pronto para GitHub Pages.

## Como jogar

1. Abra o site:
   https://thiegoortega.github.io/stoponline/
2. Digite seu nome → Enter
3. Aguarde os jogadores na tela de espera
4. Host inicia a rodada
5. Preencha Nome, País, Animal e Objeto
6. Envie respostas e veja o ranking
